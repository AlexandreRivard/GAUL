# GAUL - Groupe Aérospatial de l'Université Laval
![GAUL Banner](src/assets/gaul/GAUL_Stratos_Cover_Facebook.png)

## Project setup
```
git clone https://github.com/GuillaumeLandry/GAUL.git
```
```
cd GAUL
```
```
npm install
```

### Run a development server
```
ng serve
```

### Deploy on firebase
```
ng build --prod
```
```
firebase deploy
```
